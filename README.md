# E-Commerce Sales Analysis Using SQL

## Project Overview

This project analyzes an e-commerce sales dataset using SQL to uncover valuable business insights related to revenue, discounts, product categories, customer purchasing behavior, and payment methods.

The project demonstrates the use of SQL for data exploration, business intelligence, and decision-making through a collection of analytical queries ranging from basic aggregations to advanced window functions and Common Table Expressions (CTEs).

---

## Dataset Information

The dataset contains information about e-commerce transactions, including:

| Column Name     |        Description                 |
|-----------------|------------------------------------|
| product_id      | Unique identifier for each product |
| category        | Product category                   |
| price (₹)       | Original product price             |
| discount (%)    | Discount offered on the product    |
| final_price (₹) | Price paid after discount          |
| payment_method  | Payment method used for purchase   |
| purchase_date   | Date of transaction                |

## Data Cleaning & Validation

The dataset was validated to ensure data quality and consistency.

### Data Quality Checks Performed

1. Checked for missing (NULL) values in important columns.
2. Identified duplicate records.
3. Validated product prices to ensure no negative or zero values.
4. Verified discount percentages were within the valid range (0–100%).
5. Ensured final selling prices were logically correct.

### Sample Validation Queries

```sql
-- Check for NULL values
SELECT *
FROM ecommerce_data
WHERE product_id IS NULL
   OR category IS NULL
   OR price IS NULL
   OR final_price IS NULL;
```

```sql
-- Check for duplicate records
SELECT product_id, COUNT(*)
FROM ecommerce_data
GROUP BY product_id
HAVING COUNT(*) > 1;
```

### Validation Result

No major data quality issues were found. The dataset was clean and suitable for analysis.

## Key Findings

### 1. Sports Category Generated the Highest Revenue
The Sports category generated approximately ₹43,739 in sales, making it the top-performing category in the dataset.

### 2. UPI Was the Most Preferred Payment Method
UPI recorded the highest number of transactions and generated the highest revenue, showing strong customer preference for digital payments.

### 3. Premium Products Drove Most Revenue
Products priced above ₹300 contributed the majority of total sales revenue, highlighting the importance of premium-priced products.

### 4. Beauty Category Generated the Lowest Revenue
The Beauty category contributed the least revenue, suggesting lower demand compared to other categories.

### 5. Sports Products Received the Highest Average Discounts
The Sports category had the highest average discount percentage, indicating that promotions may have helped boost sales.

### 6. September Was the Best Sales Month
September recorded the highest monthly revenue, making it the strongest sales month in the dataset.

### 7. Revenue Was Well Distributed Across Categories
Multiple categories such as Sports, Toys, Electronics, and Books generated strong revenue, reducing dependence on a single category.

## Sample Queries Included

```sql
-- Revenue by Category
SELECT category,
       SUM(final_price) AS revenue
FROM ecommerce_data
GROUP BY category;
```

```sql
-- Revenue Ranking
SELECT category,
       SUM(final_price) AS revenue,
       DENSE_RANK() OVER(
           ORDER BY SUM(final_price) DESC
       ) AS revenue_rank
FROM ecommerce_data
GROUP BY category;
```

```sql
-- Monthly Revenue Trend
SELECT MONTH(purchase_date) AS month,
       SUM(final_price) AS revenue
FROM ecommerce_data
GROUP BY MONTH(purchase_date)
ORDER BY month;
```
